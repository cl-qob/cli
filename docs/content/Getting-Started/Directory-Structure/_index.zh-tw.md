從指令列執行 **`qob create cl-project <project-name>`** 產生器，將會建立具有下列結構的目錄：

```
.
├─ .gitignore
├─ {package-name}.asd
├─ README.markdown
├─ README.org
├─ src
│  └─ main.lisp
└─ tests
   └─ main.lisp
```

## 目錄結構說明

如需詳細資訊，請參閱 https://github.com/fukamachi/cl-project。
